01 / 三層核心結構拆解

戰鬥陀螺 X 零件示意 · 戰刃 / 輪盤 / 軸心

BLADE 戰刃（金屬打擊層）

陀螺最外圈的重裝锌合金部位。將前幾代的結晶盤與金屬環融為一體，專門負責承受與發動高強度的金屬碰撞。其不對稱形狀直接決定是攻擊型、防禦型或持久型。

RATCHET 固鎖輪盤（高度與抗爆控制）

中層的塑料改裝扣件。其外露的突起邊角被對手精準撞擊時，會觸發爆裂機制。玩家通常透過更換不同高度與刃數（如 3-60、5-80）的輪盤來調整陀螺的重心。

BIT 軸心陀尖（極限加速齒輪）

底部的物理軸芯。其軸突外圍帶有特殊齒輪，當陀螺位移到對戰盤邊緣時，齒輪會與對戰盤上的加速軌道精準咬合，瞬間爆發出超越前幾代數倍的「極限加速 (X-Dash)」。

02 / 新手零盲點入坑指南

第一步：必備專用戰鬥盤

戰鬥陀螺 X 必須在擁有外圈齒輪加速軌道的專用競技場（如 UX-04 或 BX-10 戰鬥盤）中對決。傳統的平滑舊戰鬥盤無法觸發靈魂機制「極限加速」。

# 齒輪咬合核心

第二步：首選入門套組 (Starter)

強烈建議直接購買包含發射器 + 陀螺的 Starter 組合。例如具有極高容錯率與穩定性能的「UX-01 蒼龍（Dran Buster）」或全能百搭的經典款式，能迅速建立手感。

# 性價比最高

第三步：升級拉線/拉帶發射器

標配的簡易小型發射器力量較弱。入坑後建議加購寬幅拉帶發射器（String Launcher）或強力齒條發射器，能提供更穩定的高 RPM 初速，讓陀螺展現真正實力。

# 轉速決定勝負

第四步：理解四大屬性克制

陀螺分為：攻擊（暴走對撞）、防禦（穩守中心）、持久（高續航自轉）、平衡（結合雙重優勢）。新手在挑選第二顆陀螺時，建議選擇與第一顆不同屬性的款式進行對改。
